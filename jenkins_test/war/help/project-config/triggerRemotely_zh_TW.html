如果您想要用預先定義好的特殊 URL 觸發建置作業，請啟用本功能 (方便 Script 使用)。

這個功能的典型例子是設在原始碼控制系統的 Hook Script 裡，一旦有人 Commit 變更到儲存庫就觸發建置。 也可以設在剖析原始碼控制通知信的 Script 裡。

您還要再提供一組授權用的字串 Token，只有知道的人才能遠端觸發建置本專案。
