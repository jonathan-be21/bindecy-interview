Akzeptiert eine hochzuladende Datei als Build-Parameter. Die hochgeladene Datei wird am angegebenen Speicherort im Arbeitsbereich abgelegt, von wo aus der Build-Prozess auf sie zugreifen und sie verwenden kann.

Dies ist nützlich in vielen Situationen, z.B.

  1. Ausführung von Tests auf selbstgebauten Artefakten.
  2. Automatisierung des Upload/Release/Deployment-Prozesses, bei dem der User ein File angeben kann.
  3. Jegliche Datenverarbeitung auf der hochgeladenen Datei.
