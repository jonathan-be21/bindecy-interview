Definiert einen Lauf-Parameter, bei dem der Benutzer einen bestimmten Lauf eines Projekts auswählen kann. Die absolute URL dieses Laufs kann als entweder als Umgebungsvariable oder durch Variablenersetzung in anderen Bereichen der Job-Konfiguration genutzt werden. Im Buildprozess kann so beispielsweise weitere Information über den Lauf von Jenkins abgefragt werden.
