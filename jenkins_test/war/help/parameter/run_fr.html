Définit un paramètre d'exécution, que les utilisateurs pourront choisir pour une exécution unique d'un certain projet. L'URL absolue de cette exécution sera exposée comme une variable d'environnement ou à travers la substitution de variables dans certaines parties de la configuration. Dans le build, cela peut être utilisé pour demander plus d'informations à Jenkins.
