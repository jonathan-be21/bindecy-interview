Definisce un parametro "Esegui" per cui gli utenti possono scegliere una singola esecuzione di un dato progetto. L'URL assoluto di quest'esecuzione sarà esposto come variabile d'ambiente o (per sostituzione di variabili) in altre parti della configurazione. Questo parametro può essere utilizzato durante la compilazione per richiedere ulteriori informazioni a Jenkins.
