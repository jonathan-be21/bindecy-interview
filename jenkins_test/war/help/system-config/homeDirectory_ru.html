Jenkins хранит все файлы данных внутри этой директории на файловой системе сервера. Вы можете изменить это поведение следующим образом:
  1. Используя административные инструменты веб-контейнера для установки переменной окружения JENKINS_HOME .
  2. Устанавливая переменную окружения JENKINS_HOME до запуска вашего веб-контейнера.
  3. (Не рекомендуется) Модифицируя файл web.xml в Jenkins.war (или в его распакованном образе внутри веб-контейнера)
Это значение не может быть модифицировано во время работы Jenkins. Здесь оно приведено в основном для проверки того, что ваши настройки применились.
