De nombreux aspects d'un build dépendent de l'horloge. Par conséquent, si l'horloge de la machine qui exécute Jenkins et celle de la machine agent sont très différentes, des comportements étranges peuvent se voir. Il est recommandé d'utiliser NTP pour synchroniser les horloges entre les machines.
