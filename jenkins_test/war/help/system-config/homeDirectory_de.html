Jenkins speichert alle seine Daten in diesem Verzeichnis. Sie können das Verzeichnis auf folgende Weise ändern:
  1. Setzen Sie im Administrationswerkzeug Ihres Web-Containers die Umgebungsvariable JENKINS_HOME .
  2. Setzen Sie die Umgebungsvariable JENKINS_HOME , bevor Sie Ihren Web-Container starten.
  3. (Nicht empfohlen) Editieren Sie die Datei web.xml in jenkins.war (oder dessen entpackte Entsprechung in Ihrem Web-Container).
Diese Einstellung kann nicht geändert werden, während Jenkins sich im laufenden Betrieb befindet. Diese Einstellung wird hier hauptsächlich angezeigt, damit Sie Ihre Konfiguration überprüfen können.
